# Campfire Miami Archive
A website archive and portfolio for the organizing process of Campfire Miami.

# Goal
To serve as an archive for the work done by organizers for Campfire Miami and to be used as a portfolio for the organizers to display their hard work and achomplishments!

# Image use
Headshots were sent to me from the organizers to make a post and were reused for this website
